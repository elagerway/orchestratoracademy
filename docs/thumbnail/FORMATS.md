# Thumbnail Format Definitions

## Platform Sizes

| Platform | Size | Aspect Ratio | Use |
|----------|------|-------------|-----|
| YouTube | 1280x720 | 16:9 | Video thumbnails |
| Blog | 1200x630 | ~1.9:1 | OGP images, featured images |
| X/Twitter | 1200x675 | 16:9 | Post images |

---

## Thumbnail Styles

### 1. Text-Dominant
| Item | Setting |
|------|---------|
| Trait | Delivers the message through large text |
| Best for | How-tos, list articles, major announcements |
| Layout | Text centered, simple background |
| Colors | High contrast, prioritize readability |

### 2. Person / Character
| Item | Setting |
|------|---------|
| Trait | Main visual + text |
| Best for | Interviews, reviews, case studies |
| Layout | Person on left or right, text on opposite side |
| Colors | Background that makes the person pop |

### 3. Concept
| Item | Setting |
|------|---------|
| Trait | Abstract visuals that spark curiosity |
| Best for | Technical explainers, concept intros, essays |
| Layout | Graphical elements, icons as focal point |
| Colors | Brand colors matching the theme |

### 4. Comparison
| Item | Setting |
|------|---------|
| Trait | Before/After, A vs B contrast |
| Best for | Comparison articles, transformations, showdowns |
| Layout | Left/right split, clear contrast |
| Colors | Two colors to express the contrast |

---

## Required Elements Checklist

- [ ] Catchy title (20 characters or fewer)
- [ ] High-contrast color scheme
- [ ] Highly legible font
- [ ] Clear focal point
- [ ] Readable at mobile size

---

## Color Patterns

### Attention-Grabbing Combinations
| Pattern | Main | Accent | Background |
|---------|------|--------|-----------|
| Passion | Red | White | Black |
| Trust | Blue | White | Dark navy |
| Growth | Green | White | Dark green |
| Warning | Yellow | Black | Orange |
| Cool | Purple | Pink | Black |

### Readability Principles
- Background-to-text contrast ratio of at least 4.5:1
- Add shadows or outlines to text
- Place a text backing behind text on busy backgrounds

---

## Font Usage Guide

### Recommended Font Styles
- **Heavy sans-serif**: Impact-focused
- **Rounded sans-serif**: Friendly, approachable
- **Serif**: Premium feel, authority

### Text Size Guidelines
- Main title: 15-20% of image height
- Subtitle: ~50% of main title size
- Decorative text: Small, used as accent

---

## Things to AVOID

- Text too small to read
- Too much information crammed in
- Insufficient contrast between background and text
- Too many fonts (use 2 max)
- Image too dark overall
