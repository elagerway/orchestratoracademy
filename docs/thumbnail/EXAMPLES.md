# Sample Prompt Collection

## Example 1: Text-Dominant — "How to Learn Programming"

### Prompt

```
[Format] YouTube thumbnail (1280x720)
[Title Text] "Self-Taught to Engineer in 3 Months" (placed large in center)
[Main Visual] Text is the hero element
[Background] Dark navy to purple gradient, simple
[Color Scheme] White text with yellow accent, high contrast
[Style] Text-dominant

[Layout]
- Main title centered: "Self-Taught to Engineer in 3 Months"
- Subtitle below: "— The Complete Roadmap —"
- Code icon (decorative) in bottom-right

[Text Placement]
- Main title: Center of image, heavy sans-serif, white
- "3 Months" highlighted in yellow
- Subtitle: Below main, slightly smaller, white
- Add black shadow to text

[Additional Instructions]
- Design for CTR
- Text should be large and readable
- Plenty of whitespace
- Tech-forward aesthetic
```

## Example 2: Person/Character — "React for Beginners"

### Prompt

```
[Format] YouTube thumbnail (1280x720)
[Title Text] "React for Beginners" "The Complete Guide"
[Main Visual] Young male engineer, holding a laptop and smiling, casual attire
[Background] React logo color (light blue) gradient
[Color Scheme] Light blue, white, dark blue
[Style] Person / Character

[Layout]
- Right side: Engineer character (60%)
- Left side: Title text (40%)
- Character's gaze directed toward the text
- React logo placed faintly in background

[Text Placement]
- Main title "React for Beginners": Top-left, heavy sans-serif, white
- Subtitle "The Complete Guide": Below main, slightly smaller
- React logo icon next to the title

[Additional Instructions]
- Design for CTR
- Character should have a friendly, approachable expression
- Balance between tech feel and friendliness
```

## Example 4: Comparison — "Mac vs Windows"

### Prompt

```
[Format] YouTube thumbnail (1280x720)
[Title Text] "Which Should Engineers Choose?"
[Main Visual] Left/right contrast layout, MacBook vs Windows laptop
[Background] Left side gray (Mac style), right side blue (Windows style)
[Color Scheme] Left (silver, black) vs Right (blue, white) for contrast
[Style] Comparison

[Layout]
- Left: Apple logo silhouette, MacBook
- Right: Windows logo silhouette, Windows laptop
- Center: Large "VS" (red)
- Title in question format at top

[Text Placement]
- Title "Which Should Engineers Choose?": Top center, white
- "Mac": Bottom-left
- "Windows": Bottom-right
- "VS": Large, centered, red with flame effect

[Additional Instructions]
- Design for CTR
- Create a showdown/battle feel
- Fair design that doesn't favor either side
- Tense, competitive atmosphere
```

## Example 5: Text-Dominant — "Error Resolution"

### Prompt

```
[Format] Blog thumbnail (1200x630)
[Title Text] "Fix This Error in 5 Minutes"
[Main Visual] Text is the hero, error message-style decoration
[Background] Black background, terminal-style design
[Color Scheme] Red (error), green (resolved), white text
[Style] Text-dominant

[Layout]
- Red error message-style design at top
- Main title centered: "Fix This Error in 5 Minutes"
- "5 Minutes" highlighted in green for positivity
- Terminal cursor animation-style decoration

[Text Placement]
- Error display: Top, red, monospace font
- Main title: Center, white, bold
- "5 Minutes": Green highlight
- Code-style font for tech feel

[Additional Instructions]
- Design for CTR
- Design that makes frustrated people think "This is it!"
- Convey hope for a resolution
```
